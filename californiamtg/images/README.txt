California Mortgage — hero poster / fallback image
==================================================

Drop a real cinematic California photo here:

  california-mortgage-poster.jpg   (1920x1080, optimized JPG/WebP, < 300 KB)

Then update the poster path on the <video class="hero-video"> tag in
../index.html from:

  poster="assets/california-mortgage-poster.svg"
to:
  poster="images/california-mortgage-poster.jpg"

This image is the fallback shown:
  - before the background video loads,
  - on mobile devices,
  - on slow connections,
  - if the video files are absent.

A gradient SVG placeholder (assets/california-mortgage-poster.svg) is used
until a real photo is added.
